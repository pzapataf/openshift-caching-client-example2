# Openshift Hot Rod client example
Very simple example of how to setup a Hot Rod client running inside an Openshift Online cluster.
The Openshift image will be produced by using the redhat-openjdk18-openshift STI image, by running maven and executing, copying all jars in target 
and running the main class.

## Pre-requisites
- You must have Openshift client tools installed and logged into the cluster, to retrieve service certificate.
